Quick Start

Get up and running in 30 seconds

1. Install

npm install -g project-kickoff

Or use npx without installing: npx project-kickoff init

2. Create Project

kickoff init

Answer the prompts or pick a preset. Your project is scaffolded with best practices.

3. Start Coding

cd my-project && npm run dev

Everything is configured: TypeScript, linting, Docker, CI/CD, and more.

CLI Commands

kickoff init

Interactive project generator

kickoff init --dry-run

Preview without creating files

kickoff list

Show all available options

kickoff list --verbose

Detailed descriptions

kickoff stacks [category]

Deep dive: frameworks, databases, orms, auth, ai

kickoff --help

Show all commands

ORMs & Query Builders

Bridge between your code and database. Type-safety vs flexibility.

‚öîÔ∏è Drizzle vs Prisma: The Decision Guide

Choose Drizzle when...

  • ‚úì You know SQL and want queries that look like SQL
  • ‚úì Deploying to edge/serverless (no binary engine)
  • ‚úì Bundle size matters (7KB vs Prisma's 2MB+)
  • ‚úì You want predictable queries (no magic)
  • ‚úì Performance is critical (thinner abstraction)

Choose Prisma when...

  • ‚úì You want the best developer experience
  • ‚úì Visual DB browser (Prisma Studio) is valuable
  • ‚úì Team is new to SQL/databases
  • ‚úì Complex relations and nested queries
  • ‚úì You prefer schema DSL over TypeScript schemas
Bottom line: Drizzle = SQL-first, lightweight, edge-ready. Prisma = DX-first, visual tools, higher abstraction. Both are excellent‚Äîpick based on your deployment target and SQL comfort level.
üíß

Drizzle

TypeScript
7KB Bundle SQL-like Edge-Ready

The SQL lover's ORM. Write TypeScript that reads like SQL‚Äîwhat you write is what executes. No query engine, no code generation step, no binary. Schema defined in TypeScript files. Migrations are raw SQL you can read and edit.

Why Choose This

  • Edge deployment - No binary = works on Cloudflare, Vercel Edge, Deno Deploy
  • SQL control - Queries are predictable, easy to optimize
  • Tiny footprint - ~7KB vs Prisma's 2MB+ client
  • Fast cold starts - Critical for serverless

# Project Kickoff v3.0.0 - Documentation & Polish

## Overview
Consolidate all documentation into two comprehensive HTML files and implement final polish improvements for the kickoff CLI tool.

## Goals
1. Create ONE usage guide HTML file (install, commands, troubleshooting)
2. Create ONE comprehensive stacks HTML file with ALL options from types.ts
3. Clean up duplicate docs files
4. Add vitest configuration
5. Final build verification

## Requirements

### Task 1: Create USAGE.html - Comprehensive Usage Guide
Create a single HTML file that covers:
- Installation (npm, npx)
- Quick start examples
- All CLI commands with examples:
  - kickoff create <name> [--preset] [--dry-run] [--skip-git] [-y]
  - kickoff list
  - kickoff stacks
  - kickoff presets
  - kickoff docs
  - kickoff config [--show] [--reset]
- Common workflows (SaaS app, AI app, Python API, Go service)
- Troubleshooting section:
  - Port conflicts
  - Node version requirements
  - Template errors
  - Git initialization issues
- Environment variables reference
- Next steps after project creation
